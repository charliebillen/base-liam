# BaseLiam

Like Base64 but with less characters and more Liam.

```
» echo "Sunshinnnnnnnne" | ./liamify > liamed

» cat liamed
iiLmimiiiamaimLmiaaLiaaiiamaiamaiamaiamaiamaiamaiamaiamaiaiiLLaa

» ./unliamify < liamed
Sunshinnnnnnnne
```
